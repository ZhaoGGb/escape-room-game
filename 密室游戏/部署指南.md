# 🚪 3D密室逃脱游戏 - 部署指南

## 方式一：部署到 Vercel（最简单，推荐）

### 步骤：

1. **安装 Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **登录 Vercel**
   ```bash
   vercel login
   ```

3. **部署项目**
   ```bash
   cd d:\微信小程序\密室游戏
   vercel
   ```

4. **按提示操作：**
   - 登录？Y
   - 设置范围？直接回车
   - 链接到现有项目？N
   - 项目名称：escape-room（或其他名字）
   - 目录路径：直接回车
   - 修改配置？N

5. **部署完成后**
   - Vercel会给你一个链接，如：https://escape-room.vercel.app
   - 把这个链接发给朋友就能直接玩！

---

## 方式二：部署到 GitHub Pages

### 步骤：

1. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 仓库名：escape-room-game
   - 设为 Public
   - 不要添加README（本地已有文件）

2. **上传文件**
   ```bash
   cd d:\微信小程序\密室游戏
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/escape-room-game.git
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库 → Settings → Pages
   - Source 选择 "main" 分支
   - 点击 Save

4. **获取链接**
   - 等待1-2分钟
   - 链接格式：https://你的用户名.github.io/escape-room-game/

---

## 方式三：打包成单个 HTML 文件（离线分享）

游戏已经是一个单文件，可以直接：

1. **发送 index.html 给朋友**
2. **朋友用手机浏览器打开**（需要联网加载Three.js）

---

## ⚠️ 注意事项

1. **游戏依赖网络**
   - 游戏使用 CDN 加载 Three.js 库
   - 需要联网才能玩
   - 离线版需要下载Three.js库

2. **手机访问**
   - 确保手机和电脑在同一WiFi（如果是本地测试）
   - 部署到网上后，任何设备都能访问

3. **微信内置浏览器**
   - 如果朋友用微信打开，可能需要手动复制链接到浏览器
   - 或者使用文件传输助手发送链接

---

## 🎮 游戏玩法

1. 🌱 点击盆栽 → 获得小钥匙
2. 🔑 小钥匙 → 打开抽屉 → 获得密码
3. 🔒 保险箱 → 输入密码 → 获得金钥匙+门密码
4. 🚪 金钥匙+门密码 → 逃脱成功！
